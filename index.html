Introdução

A linguagem C é uma das linguagens de programação mais importantes da história. Ela é considerada uma linguagem estruturada e de baixo nível comparada a linguagens modernas, pois permite maior controle sobre o hardware.

Foi criada na década de 1970 e até hoje é muito utilizada no desenvolvimento de sistemas operacionais, programas embarcados e também no ensino de lógica de programação.

Aprender C ajuda a entender melhor como os programas funcionam internamente, sendo uma ótima base para outras linguagens.

If Else

As estruturas condicionais são usadas para tomar decisões dentro de um programa. O if executa um bloco de código apenas se a condição for verdadeira.

O else define o que acontece quando a condição for falsa.

O else if permite verificar múltiplas condições.

  • Executa código se condição for verdadeira
  • Else executa se for falsa
if (idade >= 18) { printf("Maior de idade"); } else { printf("Menor de idade"); }
Switch

A estrutura switch é utilizada quando há várias opções possíveis para um mesmo valor. Ela torna o código mais organizado quando comparada a vários ifs encadeados.

Cada opção é representada por um case, e o comando break é usado para interromper a execução após encontrar o caso correspondente.

Também existe o default, que funciona como um caso padrão quando nenhuma das opções anteriores é atendida.

switch(opcao) { case 1: printf("Opção 1"); break; case 2: printf("Opção 2"); break; default: printf("Inválido"); }
Loops

Os laços de repetição são utilizados para executar um mesmo bloco de código várias vezes. Eles são muito úteis quando queremos automatizar tarefas repetitivas.

O for é geralmente usado quando sabemos quantas vezes o código deve se repetir. Já o while é usado quando a repetição depende de uma condição.

Essas estruturas ajudam a reduzir código repetido e tornam os programas mais eficientes e organizados.

  • Repetição de código
  • for e while são os principais

For

for(int i = 0; i < 5; i++) { printf("%d", i); }

While

while(x < 10) { x++; }
Array

Um array é uma estrutura de dados que permite armazenar vários valores do mesmo tipo em uma única variável.

Cada valor dentro do array é acessado por meio de um índice, que começa do zero. Isso facilita o armazenamento e a manipulação de grandes quantidades de dados.

Arrays são muito utilizados em situações onde precisamos trabalhar com listas de valores, como notas, números ou nomes.

int numeros[5] = {1,2,3,4,5};
  • Armazena vários valores
  • Acessado por índice
Array Multidimensional

Arrays multidimensionais são estruturas que permitem armazenar dados em mais de uma dimensão, como uma tabela ou matriz.

Eles são muito úteis para representar dados organizados em linhas e colunas, como tabelas, jogos ou sistemas mais complexos.

O acesso aos elementos é feito utilizando mais de um índice, por exemplo: matriz[linha][coluna].

int matriz[2][2] = { {1, 2}, {3, 4} };
