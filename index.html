Introdução

A linguagem C é uma linguagem de programação estruturada, ela é a principal linguagem de programação como base para as novas que surgiram até hoje, como o python por exemplo.

If / Else

As estruturas condicionais são usadas para tomar decisões dentro de um programa e conferir condições colocadas pelo desenvolvedor. Seu nome é if, que executa um bloco de código apenas se uma condição for verdadeira.

O else pode ser usado junto com o if para definir o que acontece quando a condição for falsa, ou seja, ela confere a primeira afirmação colocada pelo if e se não for verdadeira, irá executar o bloco de código seguinte, no caso, o else.

Além disso, o else if pode ser usado para verificar múltiplas condições, permitindo que diferentes blocos de código sejam executados dependendo dos valores das variáveis.
